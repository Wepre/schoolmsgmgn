# schoolmsgmgn

#### Description
c#学校人员信息管理系统，采用vs加sql，有三个用户角色，分别是教师，学生，管理员，功能简单，只实现了对数据库的增删改查
用户模块包括：
学生登录、注册、修改密码 、
教师登录、注册、修改密码。
管理员模块包括：查看用户信息(教师和学生)，添加用户、删除用户、恢复用户。
